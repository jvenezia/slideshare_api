# SlideshareApi

Slideshare ruby wrapper for the Slideshare API.

**Warning**: SlideshareApi is currently under heavy development.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'slideshare_api'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install slideshare_api
